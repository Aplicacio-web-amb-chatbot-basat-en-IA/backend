Una granja de miel funciona usando colmenas o nidos, abejas y flores cercanas para que las abejas llenen la miel con el tiempo.
Cuando la colmena esta llena, un dispensador con botellas puede recoger miel automaticamente, o con tijeras puede recoger panales.
Conviene colocar hogueras debajo si vas a recolectar manualmente para no enfadar a las abejas.
