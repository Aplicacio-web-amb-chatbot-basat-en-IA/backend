Para prepararte para pelear contra el Wither conviene llevar muy buena armadura, un arma fuerte, arco, comida abundante y pociones utiles como regeneracion o resistencia.
Tambien es recomendable llevar leche para quitar el efecto Wither y manzanas doradas para aguantar mejor el combate.
Si puedes, invocalo en un espacio controlado lejos de tu base para limitar la destruccion y tener mas control del terreno.
