Las ovejas son utiles porque dan lana y carne.

La lana es necesaria para fabricar camas y tambien es buena para decoracion.

Las ovejas pueden esquilarse con tijeras para obtener lana sin matarlas.

Criarlas con trigo permite mantener una fuente renovable de lana.

Algunas granjas avanzadas usan dispensadores con tijeras para esquila automatica.
