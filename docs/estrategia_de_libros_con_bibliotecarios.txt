Los bibliotecarios son una de las mejores fuentes de encantamientos concretos mediante comercio.

Muchos jugadores buscan libros como Mending, Efficiency, Protection, Unbreaking o Fortune.

Para conseguir una oferta deseada, suele romperse y recolocarse el atril antes del primer intercambio fijado.

Una vez se comercia con ese bibliotecario, la oferta queda fijada.

Tener una zona segura para recolocar atriles y revisar aldeanos agiliza mucho este proceso.
